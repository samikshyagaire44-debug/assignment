CSS Basics

Styling and designing beautiful web pages with Cascading Style Sheets

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation and visual formatting of HTML documents. While HTML provides the structure and content of web pages, CSS controls how that content looks, including colors, fonts, layouts, spacing, animations, and responsive behavior across different devices.

The "cascading" nature of CSS refers to how styles are applied in a hierarchical manner, where more specific rules override general ones, and later rules can override earlier ones. This allows developers to create flexible and maintainable stylesheets that can handle complex design requirements while keeping code organized.

CSS separates content from presentation, making websites easier to maintain and allowing the same HTML content to be styled differently for various contexts (desktop, mobile, print, etc.). Modern CSS includes powerful features like Flexbox, Grid, animations, and custom properties that enable sophisticated layouts and interactions.

Three Ways to Implement CSS

CSS can be applied to HTML documents in three different ways, each with its own use cases and benefits:

1. Inline CSS

CSS rules applied directly to individual HTML elements using the style attribute:

Inline CSS Example
<h1 style="color: blue; font-size: 2rem; text-align: center;">
    Styled Heading
</h1>

<p style="background-color: #f0f0f0; padding: 15px; border-radius: 5px;">
    This paragraph has inline styling applied directly to it.
</p>

<div style="width: 200px; height: 100px; background: linear-gradient(45deg, #ff6b6b, #4ecdc4);">
    Gradient background div
</div>

Use cases: Quick styling, email templates, overriding specific styles

Drawbacks: Not reusable, harder to maintain, mixes content with presentation

2. Internal CSS

CSS rules defined within the HTML document using the <style> tag in the <head> section:

Internal CSS Example
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Internal CSS Example</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            line-height: 1.6;
            margin: 0;
            padding: 20px;
            background-color: #f4f4f4;
        }
        
        .header {
            background-color: #333;
            color: white;
            text-align: center;
            padding: 1rem;
            border-radius: 8px;
        }
        
        .content {
            background-color: white;
            padding: 20px;
            margin: 20px 0;
            border-radius: 8px;
            box-shadow: 0 2px 5px rgba(0,0,0,0.1);
        }
        
        .button {
            background-color: #007bff;
            color: white;
            padding: 10px 20px;
            border: none;
            border-radius: 4px;
            cursor: pointer;
            font-size: 16px;
        }
        
        .button:hover {
            background-color: #0056b3;
        }
    </style>
</head>
<body>
    <div class="header">
        <h1>Welcome to My Site</h1>
    </div>
    
    <div class="content">
        <p>This content is styled with internal CSS.</p>
        <button class="button">Click Me</button>
    </div>
</body>
</html>

Use cases: Single-page styling, prototyping, page-specific styles

Benefits: Keeps styles with HTML, no external file needed

3. External CSS

CSS rules stored in separate .css files and linked to HTML documents:

styles.css (External File)
/* Reset and base styles */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
    line-height: 1.6;
    color: #333;
    background-color: #f8f9fa;
}

/* Header styles */
.site-header {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: white;
    padding: 2rem 0;
    text-align: center;
}

.site-header h1 {
    font-size: 2.5rem;
    margin-bottom: 0.5rem;
}

/* Navigation styles */
.nav-menu {
    display: flex;
    justify-content: center;
    list-style: none;
    gap: 2rem;
    padding: 1rem 0;
}

.nav-menu a {
    color: white;
    text-decoration: none;
    padding: 0.5rem 1rem;
    border-radius: 4px;
    transition: background-color 0.3s ease;
}

.nav-menu a:hover {
    background-color: rgba(255, 255, 255, 0.2);
}

/* Content styles */
.container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 2rem;
}

.card {
    background: white;
    border-radius: 8px;
    padding: 2rem;
    margin: 1rem 0;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
    transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.card:hover {
    transform: translateY(-5px);
    box-shadow: 0 8px 15px rgba(0, 0, 0, 0.2);
}

/* Button styles */
.btn {
    display: inline-block;
    background-color: #007bff;
    color: white;
    padding: 0.75rem 1.5rem;
    text-decoration: none;
    border-radius: 4px;
    border: none;
    cursor: pointer;
    font-size: 1rem;
    transition: all 0.3s ease;
}

.btn:hover {
    background-color: #0056b3;
    transform: translateY(-2px);
}

/* Responsive design */
@media (max-width: 768px) {
    .site-header h1 {
        font-size: 2rem;
    }
    
    .nav-menu {
        flex-direction: column;
        gap: 1rem;
    }
    
    .container {
        padding: 1rem;
    }
}
HTML File Linking External CSS
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>External CSS Example</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header class="site-header">
        <h1>My Website</h1>
        <nav>
            <ul class="nav-menu">
                <li><a href="#home">Home</a></li>
                <li><a href="#about">About</a></li>
                <li><a href="#contact">Contact</a></li>
            </ul>
        </nav>
    </header>
    
    <main class="container">
        <div class="card">
            <h2>Welcome</h2>
            <p>This page uses external CSS for styling.</p>
            <a href="#" class="btn">Learn More</a>
        </div>
    </main>
</body>
</html>

Use cases: Multi-page websites, team projects, production websites

Benefits: Reusable, maintainable, cacheable, separates concerns

Responsive Design with CSS

Modern CSS includes powerful features for creating responsive layouts that work across all devices:

Media Queries and Responsive Design
/* Mobile-first approach */
.container {
    width: 100%;
    padding: 1rem;
    margin: 0 auto;
}

/* Tablet styles */
@media (min-width: 768px) {
    .container {
        max-width: 750px;
        padding: 2rem;
    }
    
    .grid {
        display: grid;
        grid-template-columns: repeat(2, 1fr);
        gap: 2rem;
    }
}

/* Desktop styles */
@media (min-width: 1024px) {
    .container {
        max-width: 1200px;
    }
    
    .grid {
        grid-template-columns: repeat(3, 1fr);
    }
    
    .sidebar {
        display: block;
    }
}

/* Large desktop styles */
@media (min-width: 1200px) {
    .container {
        max-width: 1400px;
    }
    
    .grid {
        grid-template-columns: repeat(4, 1fr);
    }
}

/* Flexible images */
.responsive-image {
    max-width: 100%;
    height: auto;
    border-radius: 8px;
}

/* Flexible typography */
.responsive-text {
    font-size: clamp(1rem, 2.5vw, 2rem);
    line-height: 1.5;
}

CSS Best Practices

Following these best practices will help you write maintainable and efficient CSS:

  • Use external CSS files: Keep styles separate from HTML for better organization
  • Follow naming conventions: Use meaningful class names like BEM methodology
  • Organize your CSS: Group related styles together and use comments
  • Use CSS reset or normalize: Ensure consistent styling across browsers
  • Optimize for performance: Minimize CSS files and avoid redundant rules
  • Use CSS custom properties: Define reusable values with CSS variables
  • Test across browsers: Ensure compatibility with different browsers
  • Use semantic class names: Describe what elements are, not how they look
CSS Custom Properties (Variables)
/* Define custom properties in :root */
:root {
    --primary-color: #007bff;
    --secondary-color: #6c757d;
    --success-color: #28a745;
    --danger-color: #dc3545;
    --font-family-primary: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
    --font-size-base: 1rem;
    --border-radius: 6px;
    --box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
    --transition: all 0.3s ease;
}

/* Use custom properties */
.button-primary {
    background-color: var(--primary-color);
    color: white;
    font-family: var(--font-family-primary);
    font-size: var(--font-size-base);
    border-radius: var(--border-radius);
    box-shadow: var(--box-shadow);
    transition: var(--transition);
    border: none;
    padding: 0.75rem 1.5rem;
    cursor: pointer;
}

.button-primary:hover {
    background-color: color-mix(in srgb, var(--primary-color) 80%, black);
    transform: translateY(-2px);
}

.card {
    background: white;
    border-radius: var(--border-radius);
    box-shadow: var(--box-shadow);
    transition: var(--transition);
}
